<objective>
Create the executor specialist module: an isolated ExecutorState TypedDict and a
build_executor_subgraph() factory that compiles a single-node LangGraph StateGraph.
The subgraph accepts a TaskHandoff-shaped input dict, dispatches a tool from a
scoped registry, and returns an updated ExecutorState. Unit tests verify it is
invocable in isolation without the main orchestrator running.

Purpose: Phase 3's primary goal is testable isolation — each specialist must be a
self-contained compiled graph before Phase 4 wires them into the main graph.

Output: specialist_executor.py (module) + test_specialist_executor.py (tests)
