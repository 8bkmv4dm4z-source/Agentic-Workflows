## What Was Built

### Task 1: ContextManager Cross-Run Wiring

Modified `src/agentic_workflows/orchestration/langgraph/context_manager.py`:

1. **TYPE_CHECKING imports** ‚Äî `MissionContextStore` and `EmbeddingProvider` imported under `TYPE_CHECKING` to avoid circular imports at runtime.

2. **New `__init__` params** ‚Äî `mission_context_store: MissionContextStore | None = None` and `embedding_provider: EmbeddingProvider | None = None` added at the end of the signature. Stored as `self._store` and `self._embedding_provider`. Also assigns `self._logger = _logger` for use in new methods.

3. **`_persist_mission_context(mission_context: dict)`** ‚Äî Persists a completed mission to Postgres:
   - No-op if `self._store is None`
   - Extracts goal, summary, used_tools, key_results, run_id, mission_id from the dict
   - Calls `self._embedding_provider.embed_sync(goal)` if provider available, else falls back to zero vector `[0.0] * 384`
   - Calls `self._store.upsert(...)` with all fields
   - Wraps everything in `try/except` with `logger.warning` ‚Äî never raises

4. **`_get_current_goal_text(state)`** ‚Äî Extracts first goal text from `state["missions"]` list or `state["mission_contexts"]` dict.

5. **`on_mission_complete()` update** ‚Äî Builds a `persist_ctx` dict from `ctx` fields + `state.get("run_id")` and calls `self._persist_mission_context(persist_ctx)` at the end, wrapped in `try/except`.

6. **`build_planner_context_injection()` update** ‚Äî After building the base result from prior mission summaries, queries `self._store.query_cascade()` for cross-run hits, formats each as `'[Cross-run] Similar: "{goal}" ‚Üí {summary}'`, and combines with the base result up to a 1500-char total cap. Base content has priority; cross-run fills remaining capacity.

### Task 2: Unit Tests

Created `tests/unit/test_context_manager_7_3.py` with 11 tests across 3 classes:

- **TestContextManagerBackwardCompat** (3 tests): zero-arg init, None params equivalence, build_injection without store
- **TestPersistMissionContext** (4 tests): no-op with None store, upsert called with correct kwargs, embed_sync called with goal text, graceful on upsert exception
- **TestCrossRunInjection** (4 tests): `[Cross-run] Similar:` present in output, output capped at 1500 chars, no cross-run when store=None, graceful on query exception
