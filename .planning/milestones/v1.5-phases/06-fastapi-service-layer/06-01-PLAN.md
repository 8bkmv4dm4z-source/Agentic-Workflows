   - `RunRequest(BaseModel)`: user_input: str, run_id: str | None = None, prior_context: list[dict[str, Any]] | None = None
   - `RunStatusResponse(BaseModel)`: run_id: str, status: Literal["pending", "running", "completed", "failed"], elapsed_s: float | None = None, missions_completed: int = 0, tools_used_so_far: list[str] = [], result: dict[str, Any] | None = None, audit_report: dict[str, Any] | None = None, mission_reports: list[dict[str, Any]] = []

   - `SSEEvent(BaseModel)`: type: str (node_start|node_end|run_complete|state_diff|error), data: dict[str, Any]
   - `ErrorResponse(BaseModel)`: error: str, run_id: str | None = None, detail: str | None = None

4. **src/agentic_workflows/api/sse.py** — Create SSE event builder functions. Each returns a dict ready for json.dumps:
   - `make_node_start(node: str, run_id: str) -> dict` — type="node_start", tier="ui"
   - `make_node_end(node: str, run_id: str, updates: dict | None = None) -> dict` — type="node_end", tier="ui"
   - `make_run_complete(run_id: str, result: dict | None = None) -> dict` — type="run_complete", tier="ui"
   - `make_state_diff(node: str, run_id: str, diff: dict) -> dict` — type="state_diff", tier="debug"
   - `make_error(run_id: str | None, detail: str) -> dict` — type="error", tier="ui"
   All events include a `timestamp` field (ISO 8601 UTC).

5. **src/agentic_workflows/storage/__init__.py** — Empty init.

6. **src/agentic_workflows/storage/protocol.py** — Create RunStore as a Protocol class (typing.Protocol, not ABC):
   - `async def save_run(self, run_id: str, *, status: str, **fields: Any) -> None`
   - `async def get_run(self, run_id: str) -> dict[str, Any] | None`
   - `async def list_runs(self, limit: int = 50) -> list[dict[str, Any]]`
   - `async def update_run(self, run_id: str, **fields: Any) -> None`

7. **src/agentic_workflows/storage/sqlite.py** — Create SQLiteRunStore:
   - Constructor takes `db_path: str` (default from env `RUN_STORE_DB` or `.tmp/run_store.db`).
   - On init: `conn.execute("PRAGMA journal_mode=WAL")`, `conn.execute("PRAGMA busy_timeout=5000")`.
   - CREATE TABLE IF NOT EXISTS runs (run_id TEXT PRIMARY KEY, status TEXT NOT NULL DEFAULT 'pending', user_input TEXT, prior_context_json TEXT, client_ip TEXT, request_headers_json TEXT, result_json TEXT, created_at TEXT NOT NULL, completed_at TEXT, missions_completed INTEGER DEFAULT 0, tools_used_json TEXT).
   - Implements all four RunStore methods. save_run inserts, update_run uses UPDATE, get_run returns dict from Row, list_runs returns last N runs ordered by created_at DESC.
   - Use `sqlite3.connect(db_path, check_same_thread=False)` with `row_factory = sqlite3.Row`.
   - Methods are async-def but use synchronous sqlite3 calls internally (acceptable for SQLite; will be truly async with Postgres in Phase 7). Wrap sync calls in `anyio.to_thread.run_sync` for correctness.
   - Use `json.dumps(value, default=str)` for serializing RunResult and other TypedDict values.

8. **.env.example** — Add: `API_HOST=0.0.0.0`, `API_PORT=8000`, `RUN_STORE_DB=.tmp/run_store.db`
