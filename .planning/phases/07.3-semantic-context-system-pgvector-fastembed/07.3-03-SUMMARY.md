## Decisions Made

- Pool injected via `__init__`, never managed by store — matches postgres.py pattern across project
- TOOL_BITS dict append-only: bit positions frozen for existing 37 tools, new tools added at end only
- RRF k=60 standard per Cormack et al. — default parameter, overridable
- L1 short-circuit threshold: if bitmask query returns >= top_k results, skip expensive L2/L4
- `import json` deferred inside `upsert()` body to avoid import overhead in no-op pool=None paths
