## Decisions Made

- nginx:alpine as L7 LB chosen because Docker Compose deploy.replicas cannot bind the same host port across multiple replicas without it
- P1_PROVIDER=scripted in stress env ensures LLM-free deterministic responses — validates infra correctness without provider cost or latency
- pgBouncer transaction mode (not session mode) matches the project's psycopg autocommit=True pool pattern
- Scripts directory (not tests/) for stress_test.py — keeps manual tooling separate from automated pytest suite, no collection needed
